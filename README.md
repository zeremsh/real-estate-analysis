# Real Estate Market Analysis

**Tools:** Python · Pandas · NumPy · Matplotlib · Seaborn

## About
Exploratory data analysis of a real estate listings dataset.
Applied statistical methods to detect and remove outliers across 
key variables: apartment area, number of rooms, ceiling height, 
kitchen size, and listing duration.

## Key Findings
- Filtered ~15% of records as anomalous or unreliable
- Most common segment: 40–80 m² apartments with 1–3 rooms
- Apartment area and room count are the strongest price indicators
- Cleaned dataset is ready for price pr
